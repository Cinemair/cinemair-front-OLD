# Cinemair
### A Madrid open air cinema movie finder mobile app

![Cinemair](http://i.imgur.com/4wG4niB.png)

- Install ionic & gulp

```
npm install -g cordova ionic gulp
```

 - Run npm & bower

```
npm install
bower install
```

- Execute gulp

```
gulp
```

- Start ionic server

```
ionic serve
```
